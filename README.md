##  <span style="color:#00bcd4;">Descrição</span>

<span style="color:#4caf50;">Projeto simples de</span> <span style="color:#e91e63;">SlideShow em React</span> <span style="color:#4caf50;">que troca slides automaticamente e mostra o tempo em execução.</span>

---

## ⚙️ <span style="color:#ff9800;">Funcionalidades</span>

* <span style="color:#9c27b0;">Contador de tempo em segundos</span>
* <span style="color:#3f51b5;">Troca automática de slides</span>
* <span style="color:#009688;">Uso de Hooks (`useState`, `useEffect`)</span>

---

##  <span style="color:#e91e63;">Como funciona</span>

* <span style="color:#4caf50;">`useState` controla o slide atual e o tempo</span>
* <span style="color:#ff5722;">`useEffect` cria dois intervalos:</span>

  * <span style="color:#2196f3;">1s → contador de tempo</span>
  * <span style="color:#673ab7;">5s → troca de slide</span>

---

##  <span style="color:#00c853;">Como usar</span>

```javascript
import SlideShow from './SlideShow';
```

```javascript
<SlideShow />
```

---

##  <span style="color:#ff1744;">Resultado</span>

<span style="color:#795548;">Slides mudando automaticamente + tempo rodando em tempo real.</span>
